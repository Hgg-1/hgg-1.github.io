CSS样式表应用实践

探索三种CSS应用方式：外部样式表、内部样式表和内联样式

外部样式表

外部样式表将CSS代码保存在单独的.css文件中，通过<link>标签引入HTML文档。

<head>
  <link rel="stylesheet" href="styles/main.css">
</head>

这个元素样式来自外部样式表

内部样式表

内部样式表将CSS代码直接放在HTML文档的<style>标签内，通常位于<head>部分。

<head>
  <style>
    h2 {
      font-family: "楷体";
      text-decoration: underline;
      color: chocolate;
    }
  </style>
</head>

效果演示

这个元素样式来自内部样式表

内联样式

内联样式直接写在HTML元素的style属性中，仅影响单个元素。

<ul>
  <li style="color: darkorchid">
    内嵌样式表
  </li>
</ul>

作用范围 整个网站 单个HTML页面 单个HTML元素
